# Jira Blueprint

## Role

This blueprint helps draft a clear, actionable, and testable Jira ticket.

## When To Use

- To create a story, bug, or technical task.
- To turn an unclear request into an actionable ticket.
- Before running a Samourai workflow linked to Jira.

## Expected Files

- `jira-ticket.blueprint.yaml`: ticket contract.
- `jira-ticket.template.md`: description template.

## Minimal Example

```text
Prepare a Jira bug ticket for a 500 error on POST /orders.
```
